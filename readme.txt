== Description ==
Let your customer choose his favorite gift wrapping for his products. This is an easy

Current features:

* Define your own question
* Set referral locations
* Saves the location at the customer profile
* WPML compatible


= WPML compatible =
This plugin is WPML compatible. Multilingual shops can now ask where they came from in the language of the customer.

== Installation ==
1. Upload the `woocommerce-gift-wrap` folder to the `/wp-content/plugins/` directory.
1. Activate the Woocommerce Hear About Us plugin through the 'Plugins' menu in WordPress.
1. Add some referral locations in the WooCommerce Account Settings
1. Done!
